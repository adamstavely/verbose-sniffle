The Super App platform brings analytics, operations, and shared services together
in one place. This page is authored in **Markdown** — edit
`src/pages/about/overview.mdx` to change it.

## What it does

- **Analyst workspace** — search, retrieval, and document analysis.
- **Operations** — decision support, alerts, and reporting.
- **Shared platform** — authentication, data enrichment, and geospatial services.

## How this section works

Every `.mdx` file you drop into `src/pages/about/` automatically appears in the
list on the **About** page, using its `title` and `description` frontmatter. Set an
optional `order` to control where it sorts.
