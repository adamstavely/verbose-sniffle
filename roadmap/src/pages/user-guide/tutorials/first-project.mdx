By the end of this tutorial you'll have a project set up and shared with a
teammate. Follow the steps in order — each builds on the last.

## Create the project

From your workspace, choose **New project**, give it a name, and pick a template.
The starter template is a good default for your first run.

## Add your first item

Open the project and add an item. Everything you add is saved automatically and
visible to everyone with access.

## Share it

Invite a teammate from the project's **Share** menu. They'll get access
immediately — no approval step for members of your workspace.
