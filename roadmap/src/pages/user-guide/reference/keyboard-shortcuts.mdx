Shortcuts work anywhere except inside text fields.

## Global

- **/** — focus search
- **g then h** — go to Home
- **g then s** — go to Status
- **?** — open this shortcuts list

## Within a project

- **n** — new item
- **e** — edit the selected item
- **Esc** — close the current panel
